## Description

OpenInternetCryptographyKeys is a crate/library for easy, modern cryptography and abstracts from all the complicated stuff. It is serialized into PEM format.

## Algorithms

- [X] ShulginSigning
- [X] AbsolveSigning
- [X] EsphandSigning
- [X] ED25519
- [X] ED448
- [X] ECDSA (SECP256k1)
- [X] FALCON1024
- [X] SPHINCS+ (SHAKE256)
- [X] ML-DSA3 (Dilithium65)
- [X] BLS12-381
- [X] Schnorr over Ristretto
- [ ] RSA
